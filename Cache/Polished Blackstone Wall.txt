'''磨制黑石墙（Polished Blackstone Wall）'''是[[磨制黑石]]制作的[[墙]]。

== 生成 ==
磨制黑石墙不会生成在世界中。

== 获取 ==

}}

{{Stonecutting

}}

== 破坏 ==
挖掘磨制黑石墙的合适工具是镐。磨制黑石墙需要木镐或品质更好的镐来挖掘，否则挖掘时间将延长且不会掉落。

{{Breaking table}}

磨制黑石墙被破坏后会掉落自身。

== 用途 ==
{{main|墙#用途}}
磨制黑石墙可用于装饰，并有与其他墙相似的特性。
