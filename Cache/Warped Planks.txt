'''木板（Planks）'''是一类在众多[[合成]]配方中常用的[[方块]]。

'''竹板（Bamboo Planks）'''是由竹块制成的方块，功能上与木板类似。

== 生成 ==
红树木板、樱花木板、苍白橡木木板、竹板、绯红木板、诡异木板不会生成在世界中。

=== 自然生成 ===
; {{BlockSprite|Oak Planks|text=橡木木板}}
橡木木板作为以下结构的一部分生成：
* [[废弃矿井]]（作为[[橡木栅栏]]所支撑的横梁以及矿井穿过空气时的地板）
* 平原[[村庄]]（作为几种结构的墙壁和屋顶、许多建筑的地板以及水上的小径）
* [[要塞]]（作为图书馆和储藏室的地板和柱子）
* [[林地府邸]]
* [[沉船]]

; {{BlockSprite|Birch Planks|text=白桦木板}}
白桦木板作为以下结构的一部分生成：
* [[林地府邸]]
* [[沉船]]
* [[掠夺者前哨站]]

; {{BlockSprite|Spruce Planks|text=云杉木板}}
云杉木板作为以下结构的一部分生成：
* [[沼泽小屋]]（作为主要构成材料）
* [[针叶林]]村庄
* [[雪原]]与[[积雪针叶林]]村庄中的一些房屋建筑
* [[海底废墟]]
* [[沉船]]

; {{BlockSprite|Jungle Planks|text=丛林木板}}
丛林木板作为以下结构的一部分生成：
* [[沉船]]

; {{BlockSprite|Acacia Planks|text=金合欢木板}} 
金合欢木板作为以下结构的一部分生成：
* [[热带草原]]村庄
* {{only|for=[[沉船]]|bedrock}}

; {{BlockSprite|Dark oak Planks|text=深色橡木木板}} 
深色橡木木板作为以下结构的一部分生成：
* [[恶地]]生物群系里中的[[废弃矿井]]
* [[林地府邸]]
* [[海底废墟]]
* [[沉船]]
* [[掠夺者前哨站]]
* [[远古城市]]

== 用途 ==
[[File:Planks_Chart.png|不同种类木板颜色的过渡|缩略图]]

木板可用于建筑、合成和烧炼物品。

=== 合成材料 ===
木板在许多合成配方中都会被用到。

[[门]]、[[栅栏]]、[[栅栏门]]、[[台阶]]、[[楼梯]]、[[船]]、[[活板门]]、[[压力板]]、[[按钮]]和[[告示牌]]都必须用相同类型的木板合成来获得对应木板类型的方块。

其他成品不同类型的木板自由混合不会对结果造成影响。

{{crafting usage|Any Planks, Matching Planks}}

=== 燃料 ===
非下界木板可用作[[熔炉]]、[[高炉]]或[[烟熏炉]]的燃料，每个木板可烧炼1.5个物品。

=== 修复 ===
木质工具（[[木剑]]、[[木镐]]、[[木斧]]、[[木锄]]、[[木锹]]和[[木矛]]）以及盾牌可在[[铁砧]]中用木板[[修复]]。

== 你知道吗 ==
* [[Dinnerbone]]曾经开玩笑说要加入[[染色木板]]。
* 大多数可由任意木板制作的方块和物品都使用橡木木板纹理，[[物品展示框]]和[[制箭台]]使用白桦木板纹理，[[制图台]]使用深色橡木木板纹理，[[盾牌]]和[[木桶]]使用云杉木板纹理。
* 1个[[原木]]或[[木头]]可作为[[熔炉]]燃料烧炼1.5个物品，但将其分解成4个木板后则可以烧炼6个物品。
