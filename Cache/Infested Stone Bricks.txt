== 生成 ==
=== 自然生成 ===
; 虫蚀石砖、虫蚀苔石砖、虫蚀裂纹石砖、虫蚀雕纹石砖
: 虫蚀石砖可以在[[要塞]]代替普通石砖生成。要塞在生成时没有虫蚀苔石砖、虫蚀裂纹石砖或虫蚀雕纹石砖。
: 虫蚀石砖、虫蚀苔石砖、虫蚀裂纹石砖和虫蚀雕纹石砖会在[[雪屋]]地下室里生成。
; 虫蚀石头、虫蚀深板岩
: 虫蚀石头会在[[风袭丘陵]]、[[风袭森林]]、[[风袭沙砾丘陵]]和所有[[山地]]类生物群系所在的区块中生成。在Y=63以下的区域，每个这样的区块尝试生成14次虫蚀石头的[[团簇]]，大小为0-9个方块。此类生成还可以覆盖其他已生成的虫蚀方块。
: 虫蚀石头能够取代[[石头]]、[[花岗岩]]、[[安山岩]]、[[闪长岩]]、[[凝灰岩]]或[[深板岩]]生成。其中，取代凝灰岩和深板岩生成时生成的是虫蚀深板岩。
; 虫蚀圆石
: 虫蚀圆石会在[[林地府邸]]的“[[林地府邸/结构/战利品房间#伪末地传送门房|伪末地传送门房间]]”里生成。

{{Block distribution}}

=== 蠹虫 ===
虫蚀方块会在[[蠹虫]]进入对应的普通方块时生成。

== 获取 ==
{{see also|Tutorial:获取任意物品}}
虫蚀方块无法在未开启[[作弊]]的[[生存模式]]的世界中获取。使用[[创造模式]]或启用[[作弊]]，虫蚀方块可以通过[[创造模式物品栏]]、[[创造模式]]中使用{{ctrl|选取方块}}或[[命令]]等方式获得。

== 破坏 ==
挖掘虫蚀方块的合适工具是镐。虫蚀方块的挖掘用时是对应非虫蚀方块的一半，且没有对工具品质的需求。

{{Breaking table}}

没有被具有[[精准采集]]的工具挖掘破坏时，虫蚀方块被破坏后会生成一只蠹虫。

被具有[[精准采集]]的工具挖掘破坏时，虫蚀方块被破坏后会掉落对应的正常方块，不会生成蠹虫。

== 用途 ==
{{main|蠹虫#行为}}
虫蚀方块被破坏后会生成一只蠹虫。蠹虫受到攻击或[[中毒]]等[[魔法伤害]]时会使其附近21×11×21的虫蚀方块自动破坏，以生成其他蠹虫。

如果关闭游戏规则{{gamerule|doTileDrops}}，则虫蚀方块被破坏后不会产生蠹虫。如果关闭游戏规则{{gamerule|mobGriefing}}，蠹虫呼-{}-叫增援时虫蚀方块不会被破坏或生成增援的蠹虫，但会转变为对应的未虫蚀方块；且蠹虫不会钻入方块中。

=== 区别 ===
虫蚀方块与其对应的石质方块有以下区别，据此可以辨别虫蚀方块。
* 虫蚀方块的挖掘用时是对应非虫蚀方块的一半，且不使用镐挖掘时也不会延长挖掘时间。
* 放置在虫蚀方块上的[[音符盒]]将发出{{only|je|for=钢琴声}}或{{only|be|for=长笛声}}，而放置在普通的石头、圆石或石砖上的音符盒将发出底鼓声。
* [[苔藓块]]不能转化虫蚀方块。
* [[幽匿催发体]]无法将虫蚀方块转化为[[幽匿块]]，而只会在其表面生成[[幽匿脉络]]。
