[[File:Warped Forest Overworld.png|333px|缩略图|使用自定义生成设置在[[主世界]]生成的[[诡异森林]]生物群系]]
'''自定义（Custom）'''是一种[[世界类型]]，允许玩家更改主世界、下界和末地的生成设置并添加新维度，可以通过修改在世界选项界面中导出的[[JSON]]文件来实现。

== 访问 ==
自定义世界生成需使用JSON文件进行，创建世界时可以导入这些文件。玩家也可以随时从已有世界中导出这些JSON文件。

在[[创建新的世界]]菜单中，进入“更多选项”界面，点击“导入世界”按钮，选择用于自定义世界生成的JSON文件。如果JSON文件中包含有非默认的生成设置，那么此时会弹出一个警告弹窗，警告该功能目前还处于'''实验性阶段'''；点击“是”以继续，然后设置界面中世界类型变成了不可更改的“自定义”，即导入设置成功。

在世界的“编辑世界”屏幕，可以选择导出设置，设置文件位于存档的{{filepath|worldgen_settings_export.json}}下。

访问自行添加的维度的唯一方法是通过命令，比如{{cmd|execute in <维度名称> run tp @s ~ ~ ~}}，这会将玩家传送到指定的维度。

== JSON格式 ==
{{main|自定义世界生成}}
自定义世界的存储格式如下：

<div class="treeview">
* {{nbt|compound}}：根标签。
**{{nbt|boolean|bonus_chest}}：决定该世界是否会生成[[奖励箱]]。等同于创建世界时的“奖励箱”选项。
** {{nbt|boolean|generate_features}}：决定该世界是否会自然生成[[结构]]。等同于创建世界时的“{{tp|selectWorld.mapFeatures}}”选项。
** {{nbt|long|seed}}：该世界的[[种子（世界生成）|种子]]。
** {{Nbt|string|legacy_custom_options}}：[[自定义/Java版1.13前|旧的自定义世界]]的预设码，只存在于旧的自定义世界中。
** {{nbt|compound|dimensions}}：此世界中维度的列表，玩家会以{{cd|minecraft:overworld}}（主世界）作为游戏的起点。如果没有定义{{cd|minecraft:overworld}}，所有自定义维度设置都会被重置并重新生成默认的主世界。如果没有定义{{cd|minecraft:the_nether}}（下界）或{{cd|minecraft:the_end}}（末地）维度，那么[[下界传送门]]和[[末地传送门]]仍然可以被正常创建，但无法传送任何[[实体]]。
*** {{nbt|compound|<''维度命名空间ID''>}}：一个维度。世界正常运转必须要有维度{{cd|d=pretty|minecraft:overworld|minecraft:the_nether|minecraft:the_end}}。
**** {{nbt|string|type}}：[[维度类型]]命名空间ID，可引用数据包内自定义的维度类型。默认可选值有{{code|d=pretty|minecraft:overworld|minecraft:the_nether|minecraft:the_end|minecraft:overworld_caves}}。
**** {{nbt|compound|generator}}：该维度的生成器设置。
***** {{nbt|string|type|required=1|existonsave=1}}：区块生成器命名空间ID。取值只能为<code>debug</code>（[[调试模式]]）、<code>flat</code>（[[超平坦]]）或<code>noise</code>（噪声生成器）。
***** {{nbt condition|当区块生成器类型为<code>debug</code>时，没有附加字段。}}
***** {{nbt condition|当区块生成器类型为<code>flat</code>时，附加字段如下：}}
***** {{nbt|compound|settings|required=1|existonsave=1}}：超平坦设置。
****** {{nbt inherit/flat generator settings}}
***** {{nbt condition|当区块生成器类型为<code>noise</code>时，附加字段如下：}}
***** {{nbt|compound|biome_source|required=1|existonsave=1}}：设置世界生成使用的生物群系分布。
****** {{nbt|string|type|required=1|existonsave=1}}：[[世界生成#生成生物群系|生物群系源]]命名空间ID。取值只能为或<code>multi_noise</code>（多噪声型）、<code>the_end</code>（末地型）、<code>fixed</code>（单一型）或<code>checkerboard</code>（棋盘型）。
****** {{nbt condition|当生物群系源类型为<code>multi_noise</code>时，二选一，附加字段如下：}}
****** {{nbt|string}}{{nbt|compound|preset|required=1|existonsave=1}}：使用[[维度数据格式#多噪声参数列表|多噪声生物群系源列表]]生成生物群系，命名空间ID或内联定义均可。
****** {{nbt|list|biomes|required=1|existonsave=1}}：（不能为空）一个生物群系与其放置参数的列表。
******* {{nbt|compound}}
******** {{nbt|string|biome|required=1|existonsave=1}}：此参数要放置的生物群系的ID。
******** {{nbt|compound|parameters|required=1|existonsave=1}}：放置此生物群系的参数。
********* {{nbt inherit/parameter point}}
****** {{nbt condition|当生物群系源类型为<code>the_end</code>时，没有附加字段。}}
****** {{nbt condition|当生物群系源类型为<code>checkboard</code>时，附加字段如下：}}
****** {{nbt|string}}{{nbt|list|biomes|required=1|existonsave=1}}：沿着对角线重复的生物群系列表。可以为生物群系的ID、生物群系ID的列表，或一个生物群系标签。
****** {{nbt|int|scale}}：（{{interval|left=0|right=62}}，默认为2）在指数尺度上确定正方形的大小。
****** {{nbt condition|当生物群系源类型为<code>fixed</code>时，附加字段如下：}}
****** {{nbt|string|biome|required=1|existonsave=1}}：一个生物群系ID。
***** {{nbt|string}}{{nbt|compound|settings|required=1|existonsave=1}}：[[噪声设置]]，命名空间ID或内联定义均可。
</div>

=== “简洁”调试世界 ===
[[File:Lonewolf_Dimension_Commandline.png|缩略图|不存在其他维度]]
一个不包含下界和末地的[[调试模式]]世界。当玩家或其他实体触碰到[[下界传送门方块]]或[[末地传送门方块]]时也不会离开当前维度。
{{collapse|title=JSON内容|content=
<syntaxhighlight lang="json" style="display:table;">
{
  "seed": 0,
  "bonus_chest": false,
  "generate_features": false,
  "dimensions": {
    "minecraft:overworld": {
      "type": "minecraft:overworld",
      "generator": {
        "type": "minecraft:debug"
      }
    }
  }
}
</syntaxhighlight>
}}

=== 自定义超平坦维度 ===
该维度是一个由五层玄武岩、四层砂土和一层草方块由下至上依次覆盖而成的[[超平坦]]世界，整个世界只会生成村庄结构。
{{collapse|title={{cd|dimension}}对象数据|content=
<syntaxhighlight lang="json" style="display:table;">
{
  "type": "minecraft:overworld",
  "generator": {
    "type": "minecraft:flat",
    "settings": {
      "structure_overrides": [
        "minecraft:villages"
      ],
      "layers": [
        {
          "height": 5,
          "block": "minecraft:basalt"
        },
        {
          "height": 4,
          "block": "minecraft:coarse_dirt"
        },
        {
          "height": 1,
          "block": "minecraft:grass_block"
        }
      ],
      "biome": "minecraft:plains"
    }
  }
}
</syntaxhighlight>
}}
