'''阳光探测器（Daylight Detector）'''是能发出和阳光强度有关的红石信号的方块。

== 生成 ==
阳光探测器不会生成在世界中。

== 破坏 ==
挖掘阳光探测器的合适工具是斧。

{{Breaking table}}

阳光探测器被破坏后会掉落自身。

== 用途 ==
阳光探测器高0.375格（{{frac|3|8}}格），不会被[[水]]和[[熔岩]]摧毁。

{{in|je}}阳光探测器无法被[[活塞]]移动。

=== 红石元件 ===
[[File:Daylight sensor output clear.png|thumb|阳光探测器在晴天里的输出信号图表]]
{{see also|红石电路}}

阳光探测器可用来产生与[[亮度#内部光照等级|内部天空光照等级]]、天气和时间相关的红石信号。

; 输出信号
常规的阳光探测器在暴露在阳光下时输出红石信号，阳光强度越高输出强度越高。

反向阳光探测器在暴露在足够低的阳光时输出信号，阳光强度越低输出强度越高。

在下界和末地中，阳光探测器的输出固定。在常规模式下为0，在反向模式下为{{only|je|for=15}}或{{only|be|for=0}}。

光源（[[火把]]、[[荧石]]等）发出的光不影响阳光探测器。

; 行为
阳光探测器输出信号时会：
* 激活毗邻的[[红石线]]和四周输入端朝向阳光探测器的[[红石中继器]]和[[红石比较器]]，强度取决于所处位置的[[亮度#内部光照等级|内部天空光照等级]]。
* 激活毗邻的[[红石元件#机械元件|红石元件]]。如[[活塞]]、[[红石灯]]等。

阳光探测器不会充能毗邻方块。

{{in|je}}游戏每20游戏刻更新一次已加载区块中的阳光探测器的输出信号；{{in|be}}则每红石刻更新。

转换模式时阳光探测器会即时更新一次信号。方块更新不影响阳光探测器。

==== 反向阳光探测器 ====
对阳光探测器按下{{control|使用}}键会转换它的模式。反向阳光探测器输出的信号强度为15减去所在位置的[[亮度#内部光照等级|内部天空光照等级]]。若大于15或小于0，则相应地取15或0作为输出强度。

若无遮挡，反向阳光探测器在午夜晴天时输出的强度为11。天空光照减少会增加信号的强度，且会防止信号强度降到0。

{{in|je}}阳光探测器与反向阳光探测器的区别只是方块状态的不同。因此反向阳光探测器不能以物品实体形式掉落，挖掘反向阳光探测器只会掉落常规的阳光探测器。

{{in|be}}阳光探测器和反向阳光探测器是两个不同的方块。在创造模式中使用[[选取方块]]可以直接获得反向阳光探测器，但破坏反向阳光探测器只能获得阳光探测器。

=== 燃料 ===
阳光探测器可作为[[熔炉]]、[[高炉]]或[[烟熏炉]]的[[燃料]]使用，一个阳光探测器可烧炼1.5个物品。
