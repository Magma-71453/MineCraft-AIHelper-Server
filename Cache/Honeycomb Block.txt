'''蜜脾块（Honeycomb Block）'''是用[[蜜脾]]合成的装饰性方块。

== 生成 ==
蜜脾块不会生成在世界中。

== 获取 ==
=== 合成 ===
{{Crafting
|A1= Honeycomb |A2= Honeycomb
|B1= Honeycomb |B2= Honeycomb
|Output= Honeycomb Block
|type= Decoration block
}}

== 破坏 ==
没有工具可以加速对蜜脾块的挖掘。

{{Breaking table}}

蜜脾块被破坏后会掉落自身。

== 用途 ==
蜜脾块目前仅用于装饰。

== 你知道吗 ==
* 现实当中的蜜脾结构是[[wzh:正六邊形鑲嵌|正六边形镶嵌]]而成的，而蜜脾块的纹理结构似乎是[[wzh:正方形鑲嵌|正方形镶嵌]]而成的，这有可能是为了配合游戏的方块风格而采用的设计。
