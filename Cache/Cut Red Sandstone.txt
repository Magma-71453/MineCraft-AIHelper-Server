'''切制红砂岩（Cut Red Sandstone）'''是[[红砂岩]]制作的装饰性方块。

== 生成 ==
切制红砂岩不会生成在世界中。

== 获取 ==

  |type=Building block
}}

{{Stonecutting

}}

== 破坏 ==
挖掘切制红砂岩的合适工具是镐。切制红砂岩需要木镐或品质更好的镐来挖掘，否则挖掘时间将延长且不会掉落。

{{Breaking table}}

切制红砂岩被破坏后会掉落自身。
