'''雕纹石英块（Chiseled Quartz Block）'''是[[石英块]]制作的装饰性方块。

== 生成 ==
雕纹石英块不会生成在世界中。

== 获取 ==
=== 合成 ===
{{Crafting
  |B2=Quartz Slab
  |B3=Quartz Slab
  |Output=Chiseled Quartz Block
  |type=Building block
}}

=== 切石 ===
{{Stonecutting
|Block of Quartz
|Chiseled Quartz Block
}}

== 破坏 ==
挖掘雕纹石英块的合适工具是[[镐]]。雕纹石英块需要木镐或品质更好的镐来挖掘，否则挖掘时间将延长且不会掉落。

{{Breaking table}}

雕纹石英块被破坏后会掉落自身。

== 用途 ==
雕纹石英块目前仅用于装饰。
