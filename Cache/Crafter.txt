'''合成器（Crafter）'''是一种在接收到红石信号时自动[[合成]]并丢出产物的类似[[工作台]]的[[方块]]。

== 生成 ==
合成器不会生成在世界中。

== 破坏 ==
挖掘合成器的合适工具是镐。

{{Breaking table}}

合成器被破坏后会掉落自身。

== 用途 ==
合成器是[[红石元件]]，可以合成物品或作为特殊的容器。

合成器的放置方式类似于[[活塞]]，放置后其正面（即输出口的一端）会朝向玩家。

{{in|je}}合成器不能被活塞移动。

[[水]]和[[熔岩]]流会绕开合成器。

=== 容器 ===
{{Filetr|Crafter GUI.png|thumb|250px|合成器的界面}}
{{Filetr|Slot Disabling.png|thumb|250px|点击槽位可以禁用，使物品摆放成配方的顺序}}
[[File:InsertingLeather.gif|thumb|250px|漏斗向合成器的未禁用槽位中填入皮革]]
对着合成器按下{{ctrl|使用}}可以打开其界面。合成器拥有3×3的合成方格，它们亦可用作物品储存槽位，允许多名玩家同时操作。

合成方格的每个槽位都可以被单独禁用：
* 空手单击槽位可禁用该槽位，再次点击可取消禁用。
* 尝试向已禁用的槽位放置物品时将取消禁用该槽位，同时置入物品。

[[漏斗]]、[[投掷器]]和其他合成器可以与合成器的任何一面互动，将物品转入合成器的'''未禁用'''槽位。
* 接收的物品从左上至右下依次填满空的未禁用槽位。
* 如果没有空的槽位，新物品则按左上至右下的顺序优先填入同类物品堆叠数最小的槽位。
* 若新物品未能填入合适的槽位，按容器已满处理。

漏斗和[[漏斗矿车]]从合成器中取出物品的顺序与其他容器一致。

[[红石比较器]]可以检测合成器被禁用及被物品占用的槽位数量，并输出对应强度的红石信号。

合成器界面的标题默认为“合成器”，可以在放置前使用[[铁砧]]重命名来进行更改，{{in|je}}也可以通过使用{{cmd|data}}命令修改合成器的<code>CustomName</code>标签进行更改（例如，要将坐标(0,64,0)处的合成器命名为“Alice's Crafter”，可以使用命令{{cmd|data merge block 0 64 0 {CustomName:"\"Alice's Crafter\""} }}）。

{{in|je}}合成器可以被<code>lock</code>标签上锁。如果合成器的该标签不是空白的，那么这个合成器必须使用符合该标签表示的物品谓词的物品打开。

=== 红石元件 ===
{{see also|红石电路}}

; 激活
合成器是[[红石元件#机械元件|机械元件]]，可被下列形式激活：
* 毗邻的开启的'''[[红石元件#电源|电源]]'''（例外：[[红石火把]]不会激活其所附着的合成器，[[侦测器]]只会激活指向的合成器）。
* 毗邻的'''[[红石电路/充能与激活|充能方块]]'''。
* 毗邻的指向合成器且激活的[[红石中继器]]、[[红石比较器]]或[[红石粉]]（包括合成器上方的红石粉）；毗邻但未指向合成器的红石粉不会激活合成器。

{{in|je}}与发射器和投掷器不同，合成器不受[[Tutorial:半连接性|半连接性]]影响。

{{in|be}}在同一红石刻内，电容器更新阶段前存在，但在电容器更新阶段被取消的信号无法激活合成器。

; 行为
依照合成配方在合成方格内摆放物品时，右边的输出槽会显示产物，但不能像[[工作台]]一样直接取走产物，产物也不能被漏斗提取。

被激活后，合成器会产生1个计划刻，在延时2红石刻（4游戏刻，0.2秒）后合成器会消耗合成方格中的材料合成一份产物。若配方不能合成任何物品，那么材料会保留在合成器内。

合成产物后，合成器会以[[物品（实体）|掉落物]]形式从正面掷出产物。但若合成器前方存在[[容器]]（可为另一个合成器），且容器的槽位允许接收该产物，那么产物会直接存入容器（类似于投掷器，详见{{slink|投掷器|红石元件}}）。但也与投掷器不同，若容器空间不足以容纳全部产物时，多余产物会直接被掷出。

若一个配方会在合成方格中留有物品（如合成[[蛋糕]]时留下的3个[[铁桶]]、合成[[蜂蜜块]]时留下的4个[[玻璃瓶]]、合成复制[[成书]]时留下的原本成书），那么这些物品会跟随产物一起被投掷。

合成器被激活时，两个顶点处的红石纹理会亮起并发出一个音调稍高的清脆响声。合成器成功合成物品时，其顶部的合成方格纹理也会亮起并发出清脆响声，同时敞开正面的输出口并产生[[粒子#white_smoke|白烟粒子]]。合成器的音效最远可在16格听到。合成器向容器传递物品时不发出响声。

合成器只在激活信号到达（[[红石电路#信号与脉冲|上升沿]]）时合成物品，保持激活时不会连续合成物品。若要连续合成物品，需用[[时钟电路]]反复激活合成器。

合成器是[[红石导体]]，因此可以充能合成器，被充能的合成器也能作为充能方块激活毗邻的其他元件。

若以恰当的顺序控制合成器的内容物，合成器可自动完成游戏中任意配方的合成操作。
