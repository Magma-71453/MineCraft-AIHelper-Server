'''磨制黑石按钮（Polished Blackstone Button）'''是一种可提供短暂[[红石信号]]的非[[固体方块]]，是[[磨制黑石]]制作的[[按钮]]。

== 生成 ==
磨制黑石按钮不会生成在世界中。

== 获取 ==
=== 合成 ===
{{Crafting
|Polished Blackstone
|Output=Polished Blackstone Button
|type=Redstone
}}

== 破坏 ==
挖掘磨制黑石按钮的合适工具是镐。

{{Breaking table}}

按钮被破坏后会掉落自身。

== 用途 ==
{{see also|红石元件}}

在以下情况时，磨制黑石按钮会被破坏并掉落自身：
* 磨制黑石按钮的附着方块移动、消失或被摧毁；
* {{in|je}}有[[水]]流过；
* [[活塞]]尝试推动磨制黑石按钮或把方块推到磨制黑石按钮所在的方块。

{{in|je}}如果[[熔岩]]流过磨制黑石按钮，磨制黑石按钮将会被摧毁且不会掉落物品。

=== 红石电源 ===
磨制黑石按钮可用作单稳态[[红石元件#电源|电源]]（在被打开后很快自动变成关闭状态）。

; 放置
: 对着要放置的磨制黑石按钮的面{{ctrl|放置}}即可放置磨制黑石按钮。
: 只要一个表面碰撞箱完整，磨制黑石按钮就可以附着在该表面，[[树叶]]和[[紫颂花]]除外。{{in|be}}漏斗是个例外，其上表面碰撞箱不完整但可以放置磨制黑石按钮。
: 如果将磨制黑石按钮放置在方块的顶部或底部，{{in|je|nc=1}}磨制黑石按钮可以朝向任何方向，而{{in|be|nc=1}}磨制黑石按钮总是朝向一个方向。
; 激活
: 磨制黑石按钮通常处于关闭状态。
: 玩家对着磨制黑石按钮{{ctrl|使用}}可以使之暂时打开；[[爆炸#风爆|风爆]]也能使之暂时打开。
: 磨制黑石按钮会保持打开状态10个红石刻（1秒）。
; 行为
: 当打开时，磨制黑石按钮会：
:* 激活任何毗邻的[[红石粉]]至[[红石电路#信号强度|信号强度]]15。
:* 激活任何四周的背对磨制黑石按钮的[[红石比较器]]或[[红石中继器]]至信号等级15。
:* 强充能它的附着方塊，充能等级15。
:* 激活任何毗邻的（包括上面的或下面）的[[机械元件]]（例外：活塞不会被其活塞臂朝向的电源激活）。
: 当磨制黑石按钮更改状态时，它会向其毗邻提供一个[[方块更新]]（{{In|je|nc=1}}同时提供PP更新和NC更新），以及向磨制黑石按钮附着方块的毗邻提供一个方块更新（{{In|je|nc=1}}为NC更新）。
