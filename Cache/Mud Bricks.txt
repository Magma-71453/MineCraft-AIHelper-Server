'''泥砖（Mud Bricks）'''是一种由[[泥坯]]制作的装饰性方块。

== 生成 ==
=== 自然生成 ===
泥砖会生成于[[古迹废墟]]中。

{{Block distribution}}

== 获取 ==
=== 合成 ===
{{Crafting
|showname= 0
|A2= Packed Mud
|B2= Packed Mud
|A3= Packed Mud
|B3= Packed Mud
|Output= Mud Bricks, 4
|type= Building block
}}

== 破坏 ==
挖掘泥砖的合适工具是镐。泥砖需要木镐或品质更好的镐来挖掘，否则挖掘时间将延长且不会掉落。

{{Breaking table}}

泥砖被破坏后会掉落自身。

== 你知道吗 ==
* 在现实世界中，[[wzh:泥砖|泥砖]]是用泥巴和稻草混合制成的，类似于Minecraft中的[[泥坯]]，混合之后加热以增加硬度。它们被用作建筑材料的历史十分悠久，至少可追溯到5000年前。
