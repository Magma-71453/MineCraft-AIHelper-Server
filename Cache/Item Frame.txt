{{quote|物品展示框在2012年10月的骇人更新中与命令方块、铁砧、南瓜派和凋灵一同加入Minecraft。物品展示框只有一个主要用途——炫耀物品。|[[Duncan Geere]]<ref>{{article|taking-inventory--item-frame|Taking Inventory: Item Frame|May 6, 2021}}</ref>||Duncan Geere Mojavatar.png}}

'''物品展示框（Item Frame）'''是一种可以把物品或方块放在其中展示的{{only|java|for=[[实体]]|short=1}}或{{only|bedrock|for=[[方块]]|short=1}}。

'''荧光物品展示框（Glow Item Frame）'''是物品展示框的增亮变种。

== 生成 ==
荧光物品展示框不会生成在世界中。

=== 自然生成 ===
[[末地船]]的宝藏室中会生成一个物品展示框，里面放着[[鞘翅]]。

== 破坏 ==
{{in|je}}由于物品展示框属于[[实体]]，其受到[[伤害]]（例如被{{ctrl|敲击}}、被会造成伤害的弹射物击中）时会被瞬间破坏。

{{in|be}}没有工具能够加速对物品展示框的挖掘。

{{Breaking table}}

物品展示框被破坏后会掉落自身。

物品展示框中有物品时，则会先取下该物品。

== 用途 ==
=== 放置 ===
[[File:Item Frame (Map screen shown).png|thumb|150px|展示部分填充的地图的物品展示框]]
当物品展示框不满足放置条件（移除支撑方块，{{only|je|for=有方块或船<ref>{{bug|MC-154651|||Fixed}}</ref>与物品展示框发生碰撞}}）时，它会{{only|be|for=立刻}}/{{only|je|for=在{{convert|100}}内}}掉落。

{{in|be}}物品展示框可以放在[[固体方块]]的侧面、顶部和底部。

{{in|je}}由于物品展示框为实体，其放置行为与大多数方块不同：
* 物品展示框可以放在固体方块的侧面、顶部和底部，也可以放置在[[红石比较器]]、[[红石中继器]]、[[幽匿脉络]]等方块的侧面。
* 物品展示框不能和任何方块的[[碰撞箱]]产生碰撞。若一个方块的碰撞箱不完整，且不会和物品展示框产生碰撞，那么物品展示框和这个方块就可以在同一个方块位置上共存。
* 物品展示框不能和其他悬挂实体（[[画]]、[[拴绳结]]和其他物品展示框）发生碰撞。一个方块位置内可以存在多个附着在不同方向上的物品展示框，因为它们不会相互碰撞。
* 若物品展示框的{{nbt|bool|Fixed}}标签为<samp>true</samp>，则无视上方规则，可以单独存在。

=== 展示 ===
手持需要展示的物品对着物品展示框按下{{ctrl|使用}}键，即可将该物品放置到框里。物品会以自身的物品模型呈现，方块模型的大小会变得与方块对应物品实体的大小相同，并且完整方块会在展示框的前后两面各出现一半。若将[[地图]]放到框内，则展示框的大小会拓展到完整的方块侧面以供展示，但其[[边界箱]]不会改变。

{{in|be}}定位器地图会显示一个代表该物品展示框位置的绿色光标，且绿色光标也会出现在玩家手持的相同的定位器地图中。在下界进行该操作能看到绿色光标向四周随机旋转。

若要展示的物品已被[[铁砧]]重命名，当玩家准星指向该物品时会显示其名字。

在[[创造模式]]中，对着有物品的展示框按下{{ctrl|选取方块}}键将会在主手得到该物品。

荧光物品展示框自带发亮效果，并且会使其展示的物品发亮。“发亮”仅反映在纹理上，不会影响光照等级。

=== 旋转 ===
{{animate|Item Frame with Item.png;Item Frame with rotated Item.png;Item Frame with Block.png;Item Frame with rotated Block.png|150px}}

对着展示框中展示的物品按下{{ctrl|使用}}键可将其顺时针旋转。
* 每次点击物品会将其旋转45°，共8种朝向。
* 每次点击地图会将其旋转90°，共4种朝向。

{{in|je}}只要物品展示框未被取下，它就会记忆最后一个放置在展示框上的物品的朝向，并作用于下一个放置在该展示框上的物品。

=== 红石比较器 ===
展示框中物品的朝向可以被[[红石比较器]]检测，根据旋转角度输出对应信号强度（1-8）的红石信号。

== 你知道吗 ==
* 物品展示框与[[末影箱]]、[[花盆]]一样，都是基于Reddit用户的创意。 <ref>{{Tweet|dinnerbone|235747041286975488}}</ref>
* {{In|Java}}<!--尚未在基岩版测试-->如果玩家通过{{Control|选取方块}}键获得物品展示框中的[[地图]]，则得到的地图的内容会与原地图同步更新，因为其[[地图#物品数据|编号]]是相同的。
* {{in|je}}物品展示框是一种使用了方块[[模型]]的实体，它的边框使用的是白桦木板的纹理。
* {{in|je}}玩家可以通过删除展板的纹理文件使物品展示框只显示边框。可以在<code>-{}-%appdata%/[[.minecraft]]/versions/*版本*/*版本*.jar（解压后）/assets/minecraft/textures/blocks/itemframe_background.png</code>找到文件。
* {{in|je}}由于物品展示框属于实体，[[旁观模式]]下的玩家可以进入物品展示框的视角。
