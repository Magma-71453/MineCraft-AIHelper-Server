'''命令方块矿车（Minecart with Command Block）'''是一种装着{{only|for=脉冲|java}}/{{only|for=循环|bedrock}}型[[命令方块]]的矿车。

== 生成 ==
命令方块矿车不会生成在世界中，但可以使用{{cmd|summon}}等命令生成。

== 获取 ==
{{see also|Tutorial:获取任意物品}}
命令方块矿车无法在未开启[[作弊]]的[[生存模式]]的世界中获取。使用[[创造模式]]或启用[[作弊]]，命令方块矿车可以通过{{only|for=[[创造模式物品栏]]（需要启用“{{tp|options.operatorItemsTab}}”，或“{{tp|itemGroup.hotbar}}”中有命令方块矿车）|JE}}、[[创造模式]]中使用{{ctrl|选取方块}}或[[命令]]等方式获得。

== 用途 ==
{{main|矿车|命令方块}}
命令方块矿车结合了[[矿车]]和[[命令方块]]的功能。

类似于命令方块，具有足够权限时在[[创造模式]]中{{control|使用}}世界中的命令方块矿车即可打开其交互界面，与命令方块基本一致，但没有“脉冲/循环/连锁”、“条件制约/不受制约”、“保持开启/红石控制”的选项。

命令方块矿车运行在激活的[[激活铁轨]]上时，将会循环执行其中的命令，频率{{only|for=为{{convert|4}}每次|je}}/{{only|for=可更改|be}}。

命令方块矿车运行在[[探测铁轨]]上时，与该探测铁轨连接的[[红石比较器]]会输出命令方块的结果。

命令方块矿车不能通过命令方块和矿车来[[合成]]，被破坏后只会掉落1个矿车。
