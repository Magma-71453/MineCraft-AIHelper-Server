'''沉重核心（Heavy Core）'''是一种从[[试炼密室]]的[[不祥宝库]]中获得的方块，可用于合成[[重锤]]。

== 生成 ==
沉重核心不会生成在世界中。

== 获取 ==
=== 箱子战利品 ===
{{LootChestItem|heavy_core}}

== 破坏 ==
挖掘沉重核心的合适工具是镐。<ref>{{bug|MC-270322|||WAI}}</ref>

{{Breaking table}}

沉重核心被破坏后会掉落自身。

== 你知道吗 ==
* [[宝库]]战利品是沉重核心唯一的可再生获取方式。这是因为宝库可以被每个玩家打开无限次数<ref>{{bug|MC-274049|||WAI}}</ref>，且[[不祥试炼钥匙]]是[[可再生资源]]。
