'''发射器（Dispenser）'''是一种可作为一个[[红石元件]]来发射物品的固体[[方块]]。

== 生成 ==
=== 自然生成 ===
[[丛林神庙]]内会自然生成2个发射器。

[[试炼密室]]中会生成多个发射器。

== 破坏 ==
挖掘发射器的合适工具是镐。发射器需要木镐或品质更好的镐来挖掘，否则挖掘时间将延长且不会掉落。

{{Breaking table}}

发射器被破坏后会掉落自身。

== 用途 ==
发射器可用作一种容器，以及一种被激活时发射物品的[[红石元件]]。

发射器的发射面可朝向任何方向，包括上方和下方。玩家放置发射器时，发射器会朝向该玩家的方向。

=== 容器 ===
{{Filetr|Dispenser GUI.png|thumb|{{el|je}}中发射器的界面}}

发射器有9个物品槽位。对发射器按下{{ctrl|使用}}可以打开发射器的界面。

[[红石比较器]]可以检测发射器存储物品的占空比。

发射器界面的标题默认为“发射器”，可以在放置前使用[[铁砧]]重命名来进行更改，{{in|je}}也可以通过使用{{cmd|data}}命令修改发射器的<code>CustomName</code>标签进行更改（例如，要把位于(0,64,0)的发射器标记为“我的发射器”，则使用命令{{cmd|data merge block 0 64 0 {CustomName:"\"我的发射器\""} }}）。

{{in|je}}发射器可以被<code>lock</code>标签上锁。如果发射器的该标签不是空白的，那么这个发射器必须使用符合该标签表示的物品谓词的物品打开。

=== 红石元件 ===
{{see also|红石电路}}

发射器可用于随机发射所装的物品。

; 激活
发射器是[[红石元件#机械元件|机械元件]]，可被下列形式激活：
* 毗邻的开启的'''[[红石元件#电源|电源]]'''（例外：[[红石火把]]不会激活其所附着的发射器，[[侦测器]]只会激活指向的发射器）。
* 毗邻的'''[[红石电路/充能与激活|充能方块]]'''。
* 毗邻的指向发射器且激活的[[红石中继器]]、[[红石比较器]]或[[红石粉]]（包括发射器上方的红石粉）；毗邻但未指向发射器的红石粉不会激活发射器。

{{in|je}}发射器还能通过[[半连接性]]激活。若上述途径对于发射器正上方的方块（种类不限，包括[[空气]]方块和非红石导体）成立，发射器会在收到方块更新（狭义）时被激活。

{{in|be}}在同一红石刻内，电容器更新阶段前存在，但在电容器更新阶段被取消的信号无法激活发射器。

; 行为
被激活后，发射器会产生1个计划刻，在延时2红石刻（4游戏刻，0.2秒）后发射一个物品。在这4游戏刻内，发射器不会响应其他红石信号。

发射器只在激活信号到达（[[红石电路#信号与脉冲|上升沿]]）时发射物品，因此保持激活时不会连续发射物品。若要连续发射物品，需用[[时钟电路]]反复激活发射器。

若发射器储存空间的多个槽位被物品占据，则会发射随机一个槽位中的物品。随机选择的过程在发射时完成，而非发射器被激活时；在这个短暂的间隔内，玩家可以移进移出发射器中的物品。

发射器的效果取决于发射的物品。多数物品会被简单沿发射器的朝向投掷，和玩家按下了{{ctrl|丢弃物品}}键类似（但不像[[投掷器]]，发射器不能填充其他容器）。其他物品在发射时的行为则有不同。以下表格列出行为有不同的物品，若无特别说明，“朝向某方块”是指发射面紧贴该方块：

当发射器发射物品时，它会发出清脆的响声并产生[[粒子|烟雾粒子]]。若激活时发射器里面为空，则清脆响声的音调稍高。发射器的音效最远可在16格外听到。
*{{in|je}}发射器发射弹射物或刷怪蛋时不会发出响声。
*{{in|be}}发射器使用部分物品（使用剪刀、放置潜影盒等）时不会发出发射器的响声。

发射器使用并减少工具耐久度也会受到[[耐久]]魔咒的影响。

发射器是[[红石导体]]，因此可以充能发射器，被充能的发射器也能作为充能方块激活毗邻的其他元件。

== 参见 ==
其他一些方块也可以用于移动物品：
* [[投掷器]]可以丢出物品或把物品输入朝向的容器。
* [[漏斗]]可把物品输入朝向的容器。
* [[冰]]和[[浮冰]]能减少物品实体移动的摩擦力。
* [[黏液块]]被活塞推动时可以给予物品实体速度。
* [[水]]和[[熔岩]]可以推动物品实体。
* [[半连接性]]在发射器上的应用。
