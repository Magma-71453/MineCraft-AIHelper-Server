'''花岗岩台阶（Granite Slab）'''是[[花岗岩]]制作的[[台阶]]。

== 生成 ==
花岗岩台阶不会生成在世界中。

== 获取 ==

|type= Building block
}}

}}

== 破坏 ==
挖掘花岗岩台阶的合适工具是镐。花岗岩台阶需要木镐或品质更好的镐来挖掘，否则挖掘时间将延长且不会掉落。

{{Breaking table}}

花岗岩台阶被破坏后会掉落自身，双台阶被破坏后会掉落两个组成自身的台阶。
