'''木按钮（Wooden Button）'''是一种可提供短暂[[红石信号]]的非[[固体方块]]。

== 生成 ==
云杉木按钮、白桦木按钮、金合欢木按钮、深色橡木按钮、红树木按钮、樱花木按钮、苍白橡木按钮、竹按钮、绯红木按钮、诡异木按钮不会生成在世界中。

=== 自然生成 ===
丛林木按钮自然生成在沙漠村庄的房子墙上。

橡木按钮会自然生成在[[试炼密室]]中的发射器上。

== 破坏 ==
挖掘木质按钮的合适工具是斧。

{{Breaking table|橡木按钮,云杉木按钮,白桦木按钮,丛林木按钮,金合欢木按钮,深色橡木按钮,红树木按钮,樱花木按钮,竹按钮,苍白橡木按钮,绯红木按钮,诡异木按钮}}

按钮被破坏后会掉落自身。

== 用途 ==
{{see also|红石元件}}

在以下情况时，木按钮会被破坏并掉落自身：
* 木按钮的附着方块移动、消失或被摧毁；
* {{in|je}}有[[水]]流过；
* [[活塞]]尝试推动木按钮或把方块推到木按钮所在的方块。

{{in|je}}如果[[熔岩]]流过木按钮，木按钮将会被摧毁且不会掉落物品。

=== 红石电源 ===
木按钮可用作单稳态[[红石元件#电源|电源]]（在被打开后很快自动变成关闭状态）。

; 放置
: 对着要放置的木按钮的面{{ctrl|放置}}即可放置木按钮。
: 只要一个表面碰撞箱完整，木按钮就可以附着在该表面，[[树叶]]和[[紫颂花]]除外。{{in|be}}漏斗是个例外，其上表面碰撞箱不完整但可以放置木按钮。
: 如果将木按钮放置在方块的顶部或底部，{{in|je|nc=1}}木按钮可以朝向任何方向，而{{in|be|nc=1}}木按钮总是朝向一个方向。
; 激活
: 木按钮通常处于关闭状态。
: 玩家对着木按钮{{ctrl|使用}}可以使之暂时打开；[[爆炸#风爆|风爆]]也能使之暂时打开。
: 射出去的[[箭]]和投掷出的[[三叉戟]]如果落在木按钮上，木按钮会一直保持打开状态，直到箭或三叉戟被拾起或在1分钟后消失；先射箭或投掷三叉戟再放置按钮，按钮不会自动打开，但手动按一次这个按钮，它就会一直保持打开，直到箭或三叉戟消失。
: 木按钮会保持打开状态15个红石刻（1.5秒）。
; 行为
: 当打开时，木按钮会：
:* 激活任何毗邻的[[红石粉]]至[[红石电路#信号强度|信号强度]]15。
:* 激活任何四周的背对木按钮的[[红石比较器]]或[[红石中继器]]至信号等级15。
:* 强充能它的附着方塊，充能等级15。
:* 激活任何毗邻的（包括上面的或下面）的[[机械元件]]（例外：活塞不会被其活塞臂朝向的电源激活）。
: 当木按钮更改状态时，它会向其毗邻提供一个[[方块更新]]（{{In|je|nc=1}}同时提供PP更新和NC更新），以及向木按钮附着方块的毗邻提供一个方块更新（{{In|je|nc=1}}为NC更新）。

=== 燃料 ===
主世界木质按钮和竹按钮可用作[[熔炉]]、[[高炉]]或[[烟熏炉]]的燃料，每个主世界木质按钮和竹按钮能烧炼{{only|java|for=0.5个物品}}或{{only|bedrock|for=1.5个物品}}。
