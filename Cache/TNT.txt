'''TNT'''是一种能产生[[爆炸]]的方块，激活后为'''被激活的TNT（Primed TNT）'''。

== 生成 ==
=== 自然生成 ===
[[沙漠神殿]]底部的陷阱竖井里会自然生成9个TNT，与自然生成的[[石头压力板]]相连。

在[[林地府邸]]的秘密房间[[林地府邸/结构/战利品房间#伪末地传送门房|伪末地传送门房]]里会自然生成2个TNT，位于[[陷阱箱]]的两侧。

== 破坏 ==
TNT被挖掘时会立刻被破坏。

{{Breaking table|TNT}}

被爆炸破坏的TNT被破坏后会被激活，且引信时间为{{convert|10}}到{{convert|30}}。

{{in|je}}方块状态<code>unstable</code>为<code>true</code>的TNT被破坏后会被激活，且引信时间为{{convert|80}}。

其他情况下，TNT被破坏后会掉落自身。

== 用途 ==
[[File:Minecraft-tnt-explosion.gif|有框|TNT[[爆炸]]的等轴截图]]

TNT有多种方式被激活。一旦TNT被激活，就会消失并在TNT方块的中心（+0.5，+0.5，+0.5）生成一个被激活的TNT[[实体]]。

TNT被以下方式激活时，会在{{convert|80}}后爆炸：
* 玩家对其{{ctrl|使用}}[[打火石]]或[[火焰弹]]。
* {{in|be}}被玩家{{ctrl|使用}}具有[[火焰附加]]魔咒的物品。
* 被[[着火]]的[[弹射物]]击中，这些弹射物会不受阻碍地穿过并激活TNT。
* 被[[红石信号]]激活（详见下文）。
* 被[[发射器]]射出。
* 被发射器中的打火石激活。
* 被[[火]]烧毁，TNT被烧毁时会被激活而不是像其他方块那样直接消失。
* {{in|je}}方块状态<code>unstable</code>为<code>true</code>时被破坏。

TNT是[[红石元件#机械元件|机械元件]]，可被下列形式激活：
* 毗邻的开启的'''[[红石元件#电源|电源]]'''（例外：[[红石火把]]不会激活其所附着的TNT，[[侦测器]]只会激活指向的TNT）。
* 毗邻的'''[[红石电路/充能与激活|充能方块]]'''。
* 毗邻的指向TNT且激活的[[红石中继器]]、[[红石比较器]]或[[红石粉]]（包括TNT上方的红石粉）；毗邻但未指向TNT的红石粉不会激活TNT。

{{in|be}}TNT不具有“晋升为信号源”属性，因此TNT不能被[[红石电路/充能与激活|充能]]，但可以[[红石粉#形状|压线]]（即便TNT不为红石导体）。

TNT被爆炸破坏时，会在{{convert|10}}到{{convert|29}}后爆炸。

若TNT是被生物来源的因素激活的（如玩家使用打火石、有生物来源的弹射物、其他有生物来源的TNT），那么被激活的TNT也会是有生物来源的，造成的爆炸伤害会视为由该生物造成（见{{slink|伤害#造成伤害的生物}}）。

== 行为 ==
被激活的TNT能够产生[[爆炸威力]]为4的[[爆炸]]。

被激活的TNT的具有实体数据{{nbt|short|fuse}}，决定了TNT距离爆炸的剩余时间。

被激活的TNT纹理会闪烁，它会在原始[[纹理]]与比前者亮度增加40%的纹理间不停切换。纹理每秒会闪烁2下。{{in|je}}其上方会出现与[[火]]相似的烟雾效果。

TNT方块被激活时，被激活的TNT会被得到0.2格/游戏刻的垂直速度，0.02格/游戏刻的随机方向的水平速度。若未被任何[[方块]]阻挡，它将会在平地上移动0.166格。当倒计时器计时完毕时，被激活的TNT就会爆炸。
* {{in|be}}被激活的TNT若未被方块支撑，那么它至多会在爆炸前往下掉77格；
* {{in|je}}被激活的TNT若不被阻挡会下落73.45格，若被上侧方块的下表面阻挡会下落79.84格。

{{in|be}}被激活的TNT的爆炸起始点位置相对于其实体坐标高0.06125格。

被激活的TNT可以被[[水]]流或[[熔岩]]流推动，它能进入[[末地传送门]]、[[下界传送门]]和[[末地折跃门]]。传送后，被激活的TNT仍会继续闪烁，直至其在传送前的加载时间与传送后的加载时间之和达到其爆炸前的倒计时时间。

{{in|je}}被激活的TNT会被[[矛]]的冲锋攻击击退，也会被带有[[击退]]魔咒的矛的戳刺攻击击退。{{in|be}}被激活的TNT不会因矛的攻击而被击退。

{{in|je}}被激活的TNT被下界传送门、末地传送门或末地折跃门传送后不会破坏[[下界传送门方块]]，但若在被传送门传送后其所在区块被卸载或服务端被关闭，那么在区块重新加载后此被激活的TNT的爆炸仍然会破坏下界传送门方块。

被激活的TNT在[[水]]、[[熔岩]]或[[含水]]的方块中爆炸时，由于此类方块爆炸抗性高达100，爆炸无法摧毁任何方块，{{only|je|for=但会伤害生物}}<ref>{{bug|MCPE-89412}}</ref>，产生冲击波，推动爆炸范围内的实体。利用这一点可以制造[[Tutorial:TNT大炮|TNT大炮]]。

{{in|be}}在[[创造模式]]飞行中的玩家仍会被激活的TNT爆炸造成的冲击波推动。

== 你知道吗 ==
* “TNT”的全称是2,4,6-三硝基甲苯（[[wzh:三硝基甲苯|'''T'''ri'''n'''itro'''t'''oluene]]）。虽然游戏内TNT的合成配方是[[火药]]和[[沙子]]，但现实中TNT是由甲苯与浓硫酸和浓硝酸的混合物长时间加热反应得到的。
* 被激活的TNT具有和[[下落的方块]]一样的尺寸。
* 在流体中的TNT爆炸时无法摧毁方块，但可通过使用[[沙子]]或[[沙砾]]等方块“覆盖”TNT进行破坏。
* 尽管打火石在冒险模式中不能点火，但它仍可以激活TNT。
* 在[[树莓派版]]，TNT默认没有任何行为，但当数据值设为1时，它会在点燃时破坏自身。
* [[矛]]的近战攻击可以击退被激活的TNT，甚至可以使用击退附魔让其被击退得更远。
