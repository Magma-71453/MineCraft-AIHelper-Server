'''基岩（Bedrock）'''是一种常见的在[[生存模式]]里无法以正常途径[[破坏]]的[[方块]]。

== 生成 ==
=== 自然生成 ===
==== 基岩层 ====
基岩崎岖不平地生成在[[主世界]]和[[下界]]最底部的5层。正常情况下，最底层全部由基岩组成，往上每层增加约{{frac|1|5}}的间隙。

基岩同样会生成在下界顶部的5层，在Y=127的顶层全部由基岩构成，往下每层增加约{{frac|1|5}}的间隙。

基岩在主世界和下界的分布都依赖于世界的[[种子（世界生成）|种子]]。

==== 末地 ====
基岩是[[返回传送门]]结构的一部分。返回传送门的状态更新会重新生成组成其结构的所有基岩。

基岩是[[黑曜石柱]]结构的一部分，每一个黑曜石柱在其顶端都有单个基岩，作为[[末地水晶]]的基座。在复活末影龙时，末地水晶的基岩基座会随着整个黑曜石柱的重新生成而一起生成。

基岩是[[末地折跃门]]结构的一部分。至多20个末地折跃门可以在杀死末影龙后生成在末地主岛周围的固定位置，这些末地折跃门在有实体进入后还会生成处于外岛的对应的末地折跃门。末地的外岛也会自然生成将实体传送至黑曜石平台的末地折跃门。

== 获取 ==
{{see also|Tutorial:获取任意物品}}
基岩无法在未开启[[作弊]]的[[生存模式]]的世界中获取。使用[[创造模式]]或启用[[作弊]]，基岩可以通过[[创造模式物品栏]]、[[创造模式]]中使用{{ctrl|选取方块}}或[[命令]]等方式获得。

== 破坏 ==
{{see also|Tutorial:穿越和破坏基岩}}
基岩不能被挖掘破坏。

{{Breaking table}}

基岩被破坏后不会掉落。<ref name="MC-93119">{{bug|MC-93119|||Fixed}}</ref>

== 用途 ==

作为不能破坏的固体方块，基岩的作用是阻止玩家抵达主世界和下界的[[虚空]]，限制玩家在下界中的活动高度，并用于[[返回传送门]]、[[末地折跃门]]和[[末地水晶]]的基座来阻止玩家破坏这些结构。

[[末影龙]]和[[凋灵]]不能破坏基岩。

基岩不能被[[活塞]]移动。

[[生物]]不能在基岩上生成。

[[末影人]]不能将搬起的方块放置在基岩上。

[[信标]]光柱可以穿过基岩，这也是信标光柱唯一能穿透的视觉不透明方块，以允许玩家在下界激活信标。

在[[末地]]的基岩上的[[火]]不会熄灭。

== 你知道吗 ==
* 基岩的生成方式使其常常将一部分方块完全封闭，甚至可以发现被包裹在基岩中的矿石。
* 部分主世界结构（如[[废弃传送门]]、[[雪屋]]等）可能会罕见地生成在世界底部，替换底层基岩的位置。
* 从基岩被加入以来的Classic版本到Alpha的一些版本为止，游戏内部拥有一个叫做<code>Rock.png</code>的基岩纹理图片。
** 这张纹理用于Classic版本中地图外层的基岩贴图，而不是基岩方块。
* 基岩外观是一个高对比度的[[石头]]。在[[纹理更新]]前，基岩和石头拥有完全一致的图案。
* [[恼鬼]]和[[末影龙]]可以穿过基岩飞行。
