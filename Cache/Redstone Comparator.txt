'''红石比较器（Redstone Comparator）'''是能使红石信号保持、比较、作差或检测特定数据值（主要是容器装载量）的[[红石元件]]方块。

== 生成 ==
=== 自然生成 ===
红石比较器会自然生成于[[远古城市]]中心的地下室中。

== 破坏 ==
红石比较器被挖掘时会立刻被破坏。

{{Breaking table}}

红石比较器被破坏后会掉落自身。

== 用途 ==
红石比较器可放置在任何上表面边缘完整的方块（包括上半[[台阶]]、倒置[[楼梯]]和[[漏斗]]）上。{{in|be}}还可以放置在[[墙]]、[[栅栏]]、竖直的[[末地烛]]、竖直的[[避雷针]]和[[饰纹陶罐]]等方块上。

红石比较器在下列情况会被破坏并掉落：
* 附着方块移动、移除或被破坏；
* {{in|je}}有[[水]]流过；
* 有[[活塞]]推动它或将方块推入其位置。

若[[熔岩]]流过其位置，红石比较器会被破坏但不会掉落。

红石比较器具有三个输入输出端口：输出端、主输入端、侧输入端（被分为左、右两个侧输入端，计算时取两个侧输入端中的最大值）。放下比较器时，比较器的主输入端朝向玩家，上表面的三角指向输出端，两侧则为侧输入端。比较器上表面后端有两根小红石火把，上表面前端则只有一根。当比较器输出信号时，后侧的红石火把会变亮，三角也会变红。前端火把的状态可以由{{ctrl|使用}}键切换：
* 关闭、熄灭（使比较器处于“比较模式”）；
* 打开、亮起（使比较器处于“作差模式”）。

红石比较器只会从主输入端和侧输入端接受信号。侧输入端只接受来自[[红石粉]]、[[红石中继器]]、{{only|je|for=[[红石块]]、[[侦测器]]<ref>https://fallenbreath.me/2020/09/17/deeply-dissecting-minecraft_3.4/</ref>、[[避雷针]]}}和其他比较器的输入。

{{in|je}}红石比较器的输入端变化时会创建一个位于{{convert|2}}后的[[计划刻]]，在该计划刻生效时'''计算并更改'''输出。
* 由于红石比较器的计划刻优先级，一般的长度小于等于2游戏刻的信号无法令红石比较器更改输出，该性质可用于过滤过短的脉冲。

{{in|be}}红石比较器属于“电容器元件”，在电容器信号计算阶段根据上游元件列表获取输入并计算输出，并在'''阶段结束时'''同步计算结果。因此：
* 红石比较器无法接收'''在上一红石刻存在，但在该红石刻的图更新阶段被阻断的信号变化'''。
* 消费者元件可在同一红石刻接收到红石比较器的新输出值。

红石比较器输出信号时，后侧的两个小红石火把会亮起，否则熄灭。

红石比较器有四大用途：保持信号强度、比较信号强度、信号强度作差及检测方块状态（主要为容器的装载量）。

=== 保持信号强度 ===
红石比较器的侧输入端没有输入时，输出的信号强度与主输入端的信号强度'''相同'''（与红石中继器增强信号强度的功能不同）。

=== 比较信号强度 ===
{{Filetr|Comparators Explained.png|比较模式的比较器|缩略图|260px}}
比较模式的红石比较器（前侧火把熄灭）会比较主输入端与侧输入端。其输出逻辑性质可用于规整信号强度。

若任意侧输入端大于主输入，红石比较器不输出信号；否则输出与主输入相同的信号强度。

表达式：<math> \text{输出} = \text{主输入} \times \left[\max(\text{左侧输入}, \;\text{右侧输入}) \leq \text{主输入}\right] </math>

如：
* 主输入端输入强度为'''8'''的信号；左侧输入端输入强度为'''13'''的信号；右侧输入端'''无信号'''（强度为0）；则输出信号的强度为<math> 8 \times \left[\max(13, \;0) \leq 8\right] = 0 </math>。
* 主输入端输入强度为'''4'''的信号；左侧输入端输入强度为'''6'''的信号；右侧输入端输入强度为'''7'''的信号；则输出信号的强度为<math> 4 \times \left[\max(6, \;7) \leq 4\right] = 0 </math>。
* 主输入端输入强度为'''9'''的信号；左侧输入端输入强度为'''5'''的信号；右侧输入端输入强度为'''2'''的信号；则输出信号的强度为<math> 9 \times \left[\max(5, \;2) \leq 9\right] = 9 </math>。
* 主输入端输入强度为'''15'''的信号；左侧输入端输入强度为'''14'''的信号；右侧输入端输入强度为'''15'''的信号；则输出信号的强度为<math> 15 \times \left[\max(14, \;15) \leq 15\right] = 15 </math>。

{{-}}

=== 信号强度作差 ===
[[File:Redstone comparator.png|thumb|后侧输入B与边侧A/C中较强的一端相减，得1。假设A或者C中有比B更大的输入，则会输出0|260px]]
作差模式的红石比较器（前侧火把亮起）会对主输入端与侧输入端作差。

输出主输入信号与两个边侧输入中较强的信号强度之差，若结果小于0则钳制为0。

表达式：<math> \text{输出} = \max\Big(\text{主输入} - \max(\text{左侧输入},\;\text{右侧输入}), \;0 \Big) </math>

如：
* 主输入端输入强度为'''4'''的信号；左侧输入端输入强度为'''6'''的信号；右侧输入端输入强度为'''7'''的信号；则输出信号的强度为<math> \max\Big(4 - \max(6, \;7), \;0 \Big) = 0 </math>。
* 主输入端输入强度为'''9'''的信号；左侧输入端输入强度为'''5'''的信号；右侧输入端输入强度为'''2'''的信号；则输出信号的强度为<math> \max\Big(9 - \max(2, \;5), \;0 \Big) = 4 </math>。

作差模式的红石比较器在红石电路中可作为高精度非门使用。

特定类型的方块{{only|java|for=和物品展示框}}<ref>{{in|je}}物品展示框是实体而非方块。</ref>可作为红石比较器的'''受测目标'''，红石比较器会获取其[[方块状态]]或[[方块实体]]，经转换后作为自身输入。

受测目标需要位于红石比较器的主输入端方向上，且与红石比较器的主输入端'''毗邻'''或'''相隔一个不受测的方块'''才能作为红石比较器的输入。
* {{in|je}}该不受测的方块需要是[[红石导体]]。
* {{in|be}}该不受测的方块可以是除了[[标靶]]、[[鐘]]及[[大懸葉草]]外的红石导体、[[活塞]]、[[銅格柵]]、[[铁链]]或[[铜链]]。

红石比较器优先将受测目标作为输入。

{{in|je}}当受测目标与红石比较器间相隔了一个红石导体，而红石导体被充能且强度不小于15时。此时红石比较器会将该强度的信号作为输入，而不会检测受测目标的状态。<ref>{{bug|MC-64394|||WAI}}</ref>

红石比较器实际上最大可寄存2,147,483,647的信号强度，可通过[[#命令方块|检测命令方块]]或检测使用命令修改过的容器来获得大于15的信号。
{{-}}

当红石比较器检测大型箱子或大型陷阱箱时，红石比较器会检测整个大型箱子（54个槽位），而不只是红石比较器后方的那半个部分。{{in|je|nc=1}}无法打开的箱子（无论是因为上方有红石导体还是有[[猫]]坐在上方）总会使红石比较器认为信号为0，不受其中物品数量的影响。潜影盒总是可以被检测，即使无法打开。

表中展示了各容器中，最大堆叠数为64的物品需要多少组才能输出指定的信号强度值。对于最大堆叠数为16的物品（[[雪球]]、[[告示牌]]、[[末影珍珠]]等），组数除以4并向上取整就可以适用。对于不可堆叠的物品，1个即为1组。

如：仅使用最大堆叠数为64的物品，需要依靠漏斗产生强度为10的信号，需要至少3组14个物品，共206个。

<div style="border: thin solid #aaa;margin: 1em 2em;padding: 0.5em 1em 0.4em;">
; 已知物品数量，求输入信号强度值
: 容器为空时，输入0。
: 容器不为空时，输入按下列公式计算：
: <div style="margin-left: 1.6em"><math>\text{槽位填充率} = \frac{\text{槽位物品数}}{\text{此物品最大堆叠数}}</math></div>
: <div style="margin-left: 1.6em"><math>\text{信号强度} = \left\lfloor 1 + 14 \times \frac{ \text{槽位填充率之和}}{\text{槽位总数}} \right\rfloor</math></div>
: '''示例：'''发射器（共9个槽位）中有300个最大堆叠至64的物品，产生的信号强度为8：
: <div style="margin-left: 1.6em"><math> \text{信号强度} = \left\lfloor 1 + \frac{\frac{300}{64}}{9} \times 14 \right\rfloor = 8 </math></div>
: '''注意：'''不可堆叠的物品总被认为填满一个槽位。

; 已知信号强度值，求物品数量
: 适用于利用红石比较器与容器配合来输出特定强度。
: 当物品最大堆叠数为64时，输出特定信号强度所需物品数量用如下算法计算：
: <div style="margin-left: 1.6em"><math> \text{所需物品数} = \max\left( \text{信号强度},\ \left\lceil \frac{\text{槽位总数} \times 64}{14} \times (\text{信号强度} - 1) \right\rceil \right) </math></div>
: '''示例：'''用熔炉（3物品栏）得到强度为9的信号，需要110物品：
: <div style="margin-left: 1.6em"><math> \text{所需物品数} = \max\left( 9,\ \left\lceil \frac{3 \times 64}{14} \times (9 - 1) \right\rceil \right) = 110 </math></div>
</div>

==== 其他 ====
一些非容器的方块或实体也可以被红石比较器检测：

; {{BlockLink|Cake}}
: 蛋糕输出的信号强度取决于其剩余量。每片输出2强度，共7片，即完整蛋糕输出14强度。

; {{BlockLink|Cauldron}}
[[File:Cauldron Redstone Strength Values.png|炼药锅的信号强度|缩略图]]
: 炼药锅输出的信号强度取决于其填充量。从完全无[[水]]或[[细雪]]到完全充满，输出分别为0、1、2和3；完全充满[[熔岩]]，输出为3。

; {{BlockLink|Composter}}
[[File:Composter Redstone Strength Values.png|堆肥桶的信号强度|thumb]]
: 堆肥桶输出的信号强度取决于其堆肥层数，每有一层增加一点信号。堆肥层数从无堆到满，输出分别为0-7，产生[[骨粉]]则输出8。

; {{Anchor|命令方块}}{{BlockLink|Command Block}}
: 命令方块存储最后执行的命令的“成功次数”，表示此命令方块最近执行成功命令的次数。“成功”的定义由[[命令]]的成功条件决定：如果聊天界面显示红色错误信息，那么命令执行不成功。
: 没有目标或者只有一个目标的命令，因为只成功一次，在执行成功后只会输出一格信号；对多个目标生效的命令，红石比较器会输出执行成功的次数，{{only|je|for=可以}}或{{only|be|for=不可以}}超过15。
: 命令方块持续存储最后执行的命令的成功数，直到再次执行命令。因此红石比较器会持续输出同一信号强度，即使命令方块已经不在激活状态（命令方块关闭不影响比较器开关）。

; {{BlockLink|id=all-copper-bulbs|Copper Bulb}}
: 铜灯点亮时输出满强度15，否则输出0。不受[[氧化]]程度影响。 

; {{BlockLink|Crafter}}
: 合成器输出的信号强度等于被禁用及被物品占用的槽位数量，可能的取值为0-9。

; {{BlockLink|End Portal Frame}}
: 末地传送门框架有[[末影之眼]]时输出满强度15，否则输出0。

; {{ItemLink|Item Frame}}
[[File:Item frame and comparator.png|红石比较器检测物品展示框的物品是否存在及其旋转方向|缩略图]]
: 红石比较器可检测物品展示框的内含物的旋转状态。若要检测物品展示框的状态，红石比较器必须位于物品展示框后方且红石比较器的主输入端必须朝向并紧贴物品展示框的附着方块。若物品展示框内无物品，则输出0，否则根据物品旋转角度输出1至8强度（1为默认方向，{{ctrl|使用}}旋转直到8，然后再返回1）。

; {{BlockLink|Jukebox}}
: 唱片机输出的信号取决于其中储存的[[音乐唱片]]。
{{Collapse|title=对应关系列表|content={{:音乐唱片/信号强度}}}}

; {{BlockLink|Lectern}}
: {{in|je}}讲台输出的信号强度取决于当前玩家打开[[书]]的页码与书的总页码之比，例如一本有15页的书每翻一页提升1信号强度，或5页的书每翻一页提升3强度。
: {{in|be}}因为讲台同时展示两页内容，讲台输出的信号强度取决于右侧的页码与书的总页码之比，若为最后一页则输出为15。

; {{BlockLink|Bee nest}}和{{BlockLink|Beehive}}
: 蜂巢和蜂箱会输出与其方块状态{{cd|honey_level}}的值相同强度的红石信号。

; {{BlockLink|Respawn Anchor}}
: 重生锚会输出其能量等级乘4再减1的值，即：1级能量对应3级信号，2级能量对应7级信号，3级能量对应11级信号，4级能量对应15级信号。

; {{BlockLink|Chiseled Bookshelf}}
: 雕纹书架会输出与其方块状态{{cd|last_interaction_book_slot}}的值（最后一次互动的槽位号）相同强度的红石信号。

; {{BlockLink|Sculk Sensor}}
: 幽匿感测体会输出与其感测到的振动频率值相同强度的红石信号，参见[[幽匿感测体/振动频率]]。

; {{BlockLink|Creaking Heart}}
: 嘎枝之心会输出与其绑定的[[嘎枝]]的距离对应的红石信号，参见{{slink|嘎枝之心|红石比较器}}。

; {{BlockLink|id=all-copper-golem-statues|Copper Golem Statue}}
: [[铜傀儡像]]会根据其姿势输出对应的红石信号。

; {{BlockLink|Shelf}}
: [[展示架]]会根据其三个栏位放置物品与否输出对应的红石信号，参见{{slink|展示架|红石比较器}}。
