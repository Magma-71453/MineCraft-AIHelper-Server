'''巨人（Giant）'''是一种有着[[僵尸]]外观的巨型未使用生物，不会自然生成。

== 生成 ==
巨人的生成要求亮度等级在8以下且在11以上，这个条件导致它们无法自然生成，因此也不能由[[刷怪笼]]生成。

可以使用{{cmd|summon minecraft:giant}}命令来生成巨人。

巨人{{in|console|nc=1}}存在刷怪蛋，但是它们的刷怪蛋只能通过编辑器放到玩家的[[物品栏]]里来获取。

== 掉落物 ==
巨人被[[玩家]]或驯服的[[狼]]杀死时掉落{{xp|5}}。

== 行为 ==
巨人身高近12米。尽管它们使用放大了6倍的[[僵尸]]模型，但它们不属于[[亡灵生物]]。

巨人跟僵尸共享一个完全相同的模型。这意味着改变僵尸的模型也会相应地使巨人的模型发生改变。

目前巨人不具备[[生物AI]]，因此巨人没有任何行为。它们在生成时默认面朝南（可以通过命令修改、或将其推入面向不同方向的矿车来改变朝向）、永远不会移动和转动身体。即使受到攻击，它们也只是被击退并产生[[玩家]]的受伤[[音效]]，不会有其他反应。

巨人的双臂不在它的[[碰撞箱]]内，因此攻击巨人该部位并不会造成伤害。

== 你知道吗 ==
* 玩家可以使一个刷怪笼生成其他种类的实体，并设定其骑着巨人出现。这样可以绕过生成条件限制生成巨人。
* [[Notch]]禁止巨人生成是因为“它们太强大了”。但他没有完全移除巨人是因为“它们实在是太酷炫了”。{{cn}}
* 巨人的攻击伤害[[属性]]基础值为50。这意味着如果为巨人添加近战攻击行为，巨人能够在简单、普通、困难难度分别造成{{AutoDmg|50|join=、|info=}}伤害。
* 在愚人节快照[[24w14potato]]中，巨人具有生物行为，自然生成于与超级马铃薯怪的战斗中，并且使用了Mojang Studios开发者[[Jay Wells]]的皮肤。
