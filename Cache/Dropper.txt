'''投掷器（Dropper）'''是一种可以丢出[[物品]]，或将物品转移到另外一个[[容器]]中的[[方块]]。

== 生成 ==
投掷器不会生成在世界中。

== 破坏 ==
挖掘投掷器的合适工具是镐。投掷器需要木镐或品质更好的镐来挖掘，否则挖掘时间将延长且不会掉落。

{{Breaking table}}

投掷器被破坏后会掉落自身。

== 用途 ==
投掷器可以作为储存物品的容器，或用作转移物品的[[红石元件]]。

若要放置投掷器，应对着方块表面{{ctrl|放置方块}}。投掷器可朝向任何一个方向，包括上方和下方。投掷器在被放置时面向玩家。

[[水]]和[[熔岩]]流会绕开投掷器。

{{in|je}}投掷器不能被活塞移动。

=== 容器 ===
投掷器有9个物品槽位。对投掷器按下{{ctrl|使用}}可打开投掷器的界面。

[[红石比较器]]可检测投掷器存储物品的占空比。

投掷器界面的标题默认为“投掷器”，可以在放置前使用[[铁砧]]重命名来进行更改，{{in|je}}也可以通过使用{{cmd|data}}命令修改投掷器的<code>CustomName</code>标签进行更改（例如，要将坐标(0,64,0)处的投掷器命名为“我的投掷器”，可以使用命令{{cmd|data merge block 0 64 0 {CustomName:"\"我的投掷器\""} }}）。

{{in|je}}投掷器可以被<code>lock</code>标签上锁。若投掷器的该标签不是空白的，那么这个投掷器必须使用符合该标签表示的物品谓词的物品打开。

=== 红石元件 ===
{{see also|红石电路}}
投掷器可以丢出物品，或将物品转移到另外一个[[容器]]中。

; 激活
投掷器是[[红石元件#机械元件|机械元件]]，可被下列形式激活：
* 毗邻的开启的'''[[红石元件#电源|电源]]'''（例外：[[红石火把]]不会激活其所附着的投掷器，[[侦测器]]只会激活指向的投掷器）。
* 毗邻的'''[[红石电路/充能与激活|充能方块]]'''。
* 毗邻的指向投掷器且激活的[[红石中继器]]、[[红石比较器]]或[[红石粉]]（包括投掷器上方的红石粉）；毗邻但未指向投掷器的红石粉不会激活投掷器。

{{in|je}}投掷器还能通过[[半连接性]]激活。若上述途径对于投掷器正上方的方块（种类不限，包括[[空气]]方块或非红石导体）成立，投掷器会在收到方块更新（狭义）时被激活。

{{in|be}}在同一红石刻内，电容器更新阶段前存在，但在电容器更新阶段被取消的信号无法激活投掷器。

; 行为
被激活后，投掷器会产生1个计划刻，在延时2红石刻（4游戏刻，0.2秒）后投掷一个物品。在这4游戏刻内，投掷器不会响应其他红石信号。

投掷器只在激活信号到达（[[红石电路#信号与脉冲|上升沿]]）时投掷物品，保持激活时不会连续投掷物品。若要连续投掷物品，需用[[时钟电路]]反复激活投掷器。

若投掷器储存空间的多个槽位被物品占据，则会投掷随机一个槽位中的物品。随机选择的过程在投掷时完成，而非投掷器被激活时；在这个短暂的间隔内，玩家可以移进移出投掷器中的物品。

若投掷器直接面向一个容器，则被投掷的物品会直接存放到容器中，若该容器已满，则物品会留在投掷器内；若投掷器未面向容器，那么投掷的物品会从投掷器正面向外掷出，与玩家使用{{ctrl|丢弃物品}}键效果相同。

当投掷器投掷物品时，它会发出清脆的响声。若激活时投掷器里面为空，则清脆响声的音调稍高。投掷器的音效最远可在16格外听到。投掷器向容器传递物品时不发出响声。

投掷器是[[红石导体]]，因此可以充能投掷器，被充能的投掷器也能作为充能方块激活毗邻的其他元件。
