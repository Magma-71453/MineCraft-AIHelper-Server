'''原木（Log）'''或'''菌柄（Stem）'''，是可以在[[树木]]或[[巨型下界菌]]上自然生成的[[方块]]。原木包括橡木原木、白桦原木、云杉原木、丛林原木、深色橡木原木、金合欢原木、红树原木、樱花原木和苍白橡木原木；菌柄包括绯红菌柄和诡异菌柄。

== 生成 ==
=== 自然生成 ===
==== 树木 ====
橡木、云杉、白桦、丛林、金合欢、深色橡木、红树、樱花和苍白橡木原木会分别作为其对应[[树木]]的枝干生成；橡木原木亦作为[[杜鹃树]]的枝干生成。

==== 巨型下界菌 ====
绯红和诡异两种菌柄会分别作为其对应[[巨型下界菌]]的主干生成。

==== 沉船 ====
{{in|je}}橡木、云杉和深色橡木原木会作为[[沉船]]的桅杆生成。

==== 其他结构 ====
; 橡木
橡木原木作为平原[[村庄]]房屋的一部分生成，也用作[[沼泽小屋]]的支柱。

橡木原木也会作为[[废弃矿井]]中支撑悬空矿道的立柱生成。

橡木原木也可能会在[[试炼密室]]的“入口”结构中作为植物造景的一部分生成。

; 云杉木 
云杉原木作为针叶林和雪原村庄房屋的一部分生成。

; 金合欢木 
金合欢原木会作为热带草原村庄房屋的一部分生成。

; 深色橡木
深色橡木原木可以在[[掠夺者前哨站]]的瞭望塔、笼子或原木堆中生成，也会作为[[林地府邸]]和[[远古城市]]的一部分生成。

深色橡木原木还会作为[[恶地]][[废弃矿井]]中支撑悬空矿道的立柱生成。

; 红树木
红树原木可能会在试炼密室的“入口”结构中作为植物造景的一部分生成。

{{Block distribution}}

=== 树苗 ===
[[树苗]]成熟后会生成对应的树，具有对应的原木作为树的枝干；橡木原木亦作为杜鹃树的枝干生成。

=== 下界菌 ===
[[下界菌]]成熟后会生成对应的[[巨型下界菌]]，具有对应的菌柄作为巨型下界菌的枝干。

== 破坏 ==
挖掘原木与菌柄的合适工具是斧。

{{Breaking table}}

原木与菌柄被破坏后会掉落自身。

== 用途 ==
对原木、菌柄{{ctrl|使用}}任意种类的[[斧]]可以将其转换为对应的去皮变种。

{{Smelting

菌柄不可烧炼制得木炭。

=== 燃料 ===
原木可以作为[[熔炉]]、[[高炉]]和[[烟熏炉]]的[[燃料]]，每块原木可以烧炼1.5个物品。

菌柄不能作为燃料使用。

=== 山羊冲撞 ===
[[山羊]]在冲撞原木时会掉落[[山羊角]]。

== 你知道吗 ==
* {{in|bedrock}}深色橡木原木和金合欢原木曾共享一个与旧原木分开的方块ID，现用作别名ID，叫作{{code|log2}}，因为之前原木的其他所有的数据值都被新的旋转变种占用（[[红砂岩台阶]]也是如此）。含剩余8个数据值的方块仍然不能获得。
* 尽管菌柄既不可燃烧也不能被烧制，但是它们做成的木质工具、木棍、工作台和箱子等物品和方块仍可以作为熔炉燃料。
* 两种菌柄都有动态的纹理，但去皮变种没有。
